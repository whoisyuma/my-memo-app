# My Memo App
PHP と MySQL を使って作成した、シンプルなメモ帳アプリです。  
メモの新規作成、一覧表示、詳細確認、削除などの基本機能を備えています。

## 使用技術
- PHP
- MySQL
- HTML / CSS

## 主な機能
- メモの新規作成
- メモの一覧表示
- メモの詳細ページ
- メモの削除
